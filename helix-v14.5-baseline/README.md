## Setup Instructions for Helix Repo

### 1. Copy Files

```bash
# Copy all baseline files to your Helix repo
cp -r helix-v14.5-baseline/* /path/to/Helix/
```

### 2. Create Required Directories

```bash
cd /path/to/Helix
mkdir -p Helix/state Helix/commands Helix/ethics Shadow/manus_archive
```

### 3. Set up Environment

```bash
cp .env.example .env
# Edit .env with your tokens
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Test Locally

```bash
python backend/main.py
# Should start FastAPI on http://localhost:8000
```

### 6. Deploy to Railway

```bash
railway init
railway link
railway up
```
